----Minha primeira API criada utilizando flask----

Primeiramente a API não está sendo rodada em um servidor privado, então caso você
queria testar a API terá que baixar o arquivo e rodar run.py assim vai conseguir
testar todas as rotas.

Lembrando que para acessar a documentação de todas as rotas aqui está o link com
todas as rotas documentadas: https://documenter.getpostman.com/view/20330486/UyxeoTd8

Eu deixei o arquivo um .db  que está na pasta BancoDados caso você queira testar
com mais facilidade
